## API Endpoints

### REST API (http://localhost:8000)
| Endpoint | Method | Description |
|----------|--------|-------------|
| `/sessions` | POST | Create new session |
| `/sessions/{id}` | GET | Get session info |
| `/sessions/{id}` | DELETE | Terminate session |
| `/chat` | POST | Send chat message |
| `/synthesize` | POST | Text-to-speech |
| `/transcribe` | POST | Speech-to-text |

### WebSocket (ws://localhost:8001)
- Audio streaming with base64 encoding
- Control messages for recording state
